What is CDN-FX?

CDN-FX is a ticket manager for your 3DS. You can view your ticket.db and download your own eShop content packed as a CIA. These can be installed on your 3DS.

What features can CDN-FX offer?

  • Show detailed info of your tickets from ticket.db
  • Sort every ticket to a Category and list them in a table
  • Displays Ticket info by getting information from the community
  • Search Function
  • Filter to find legit CIAs
  • Download Content from CDN Server
  • Creating CIAs from encTitleKeys.bin
  • Creating Tickets
  • Titledatabase Viewer
  • Patch the Demo Play Count
  • Patch DLCs to unlock all content
  • Building CIA
  • Building CIAs out of raw content (Offline Mode)
  • Set Default paths
  • Debug Mode
  • Updater

FAQ

Q: Do I need Python?

A: No, but you need Java 8.

Q: Why does it take some time to start?

A: If you have many default paths setted, it will take some time to load it.

Q: Why I can't download System Titles or Titles with ConsoleID [00000000]?

A: This is disabled by default, just enable it in the settings.

Q: Something doesn't work properly.

A: Just enable the Debug Mode in the settings and send me the debug.log file on GBAtemp.

How to use

Before you can use CDN-FX, make sure you have Java 8 installed. Then just open it by doubleclick the .jar archive. If you want to add/remove something to the downloadqueue, just do a doubleclick on a title.
