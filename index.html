What is GroovyCIA?

GroovyCIA is a ticket manager for your 3DS. You can view your ticket.db and download your own eShop content packed as a CIA. These can be installed on your 3DS.

What features can GroovyCIA offer?

  • Counting tickets from ticket.db
  • Sort every ticket to a Category and list them in a table
  • Displays Ticketinfo by getting informations from 3dsdb and from the community
  • Set your own Ticketinfo
  • Searchfunction
  • Filter to find legit CIAs
  • Downloading Content from CDN Server
  • Patching the Demoplaycount
  • Patching DLCs
  • Building CIA
  • Building CIAs out of raw content
  • Create personal CIAs
  • Set default outputfolder and ticket.db
  • Debugmode

FAQ

Q: Do I need Python?
A: No, but you need Java 8.

Q: Why does it take some time to start?
A: GroovyCIA downloads the newest 3dsdb database. To disable this just change the settings.

Q: Why I can't download Systemtitles or Titles with ConsoleID [00000000]?
A: This is disabled by default, just enable it in the settings.

Q: Something doesn't work properly.
A: Just enable the Debugmode in the settings and send me the debug.log file on GBAtemp.

How to use

Before you can use GroovyCIA, make sure you have Java 8 installed. Then just open it by doubleclick the .jar archive. If you want to add something to the downloadqueue, just do a right-click on a title and you will see a drop-down-menu.
